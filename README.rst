===========
n2n-formula
===========

A saltstack state to manage n2n (node or supernode)

Available states
================

.. contents::
    :local:

``n2n.node``
------------

Installs the n2n package and starts a n2n node.

``n2n.supernode``
-----------------

Installs the n2n package and starts a n2n supernode.
